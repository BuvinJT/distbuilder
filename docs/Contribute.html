Contact / Contribute¶

Join GitHub¶

If you wish to report bugs, make request features, or contribute to this project, please join GitHub.

Post An Issue¶

Once you have a GitHub account, you can post an issue for this project.

Contribute Code¶

If you wish to contribute code, create your own fork from distbuilder. After that, git clone your fork, and push the commits. Finally, issue a pull request. Upon review, your contribution maybe merged into the official project.
