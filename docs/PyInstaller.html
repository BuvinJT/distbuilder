    obj.injectDuplicateDataPatch(self)¶

    This patches a known bug in PyInstaller on Windows. PyInstaller analysis can build a set of data file names which contain “duplicates” due to the Windows file system case insensitivity. This patch eliminates such duplicates, thus preventing runtime errors in the binary produced.
